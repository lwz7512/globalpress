    • WorldBank Contract Awards
      We’re using the World Bank’s Major Contract Awards dataset,
      which serves as a guide on the distribution of major contract
      commitments among the Bank’s member countries.

      Version 33 at World Bank Global Development Sprint
    • What's going on
      1. • Each luminous particle is a contract. It moves from supplier to borrower.
      2. • The size of particle depends of contract amount.
      3. • Each node of country gathers contracts which it borrowed.
      4. • Every second is a day.
      5. • The histogram shows the number of contracts per day, separated by a selected criterion.
      6. • The legend shows the number of contracts for all time separated by a selected criterion.
    • How to use:
      1. • Uses buttons for controlling process visualization.
      2. • Uses select for choosing fiscal year or criteria of colouring.
      3. • Uses mouse wheel for zooming.
      4. • Uses mouse cursor for showing information about focused node.
